# AI-based Automated Resume Summarizer, Scorer and Filtration in Job Application Webpage

This project is a web-based recruitment system that automates resume collection, filtration, and candidate scoring through an integrated job applicant website, AI-based resume analysis, and an admin dashboard for HR. It helps recruiters efficiently shortlist candidates by reducing manual resume screening.

**Features**
* Job applicant career page for resume submission
* Automated resume parsing and keyword extraction
* Resume filtration and scoring based on job requirements
* Admin panel for HR to view, filter, and manage applicants
* Secure resume upload and data storage

**Technical Details**
* Backend: Python, Flask
* Frontend: HTML, CSS, JavaScript
* Resume Processing: NLP-based resume parsing and keyword matching
* Scoring Logic: Rule-based / keyword-weighted scoring mechanism
* Database: SQLite with SQLAlchemy ORM
* File Handling: Secure PDF resume upload and storage

**Use Cases**
* Company career portals
* Automated HR shortlisting systems
* Internship and job application platforms
* Resume screening for recruiters
